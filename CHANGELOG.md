# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Changed
- 首次进入 TUI 时，如果列表中存在与当前正式配置内容一致的方案，默认选中第一个 `☑` 方案
- 执行成功或按 `R` 刷新后，尽量保留当前替换方案的选中状态，不再重置到第一个
- 调整左右对比面板标题格式，确保文件内容第一行不再被页码信息挤占
- 优化对比面板标题颜色组合：
  - `[正式]` 使用青色
  - `[候选]` 使用黄色
  - `[已选]` 使用绿色
  - `[未选]` 使用红色

## [1.1.0] - 2026-03-30

### Added
- 在替换方案列表中，对“候选内容与当前正式文件内容完全一致”的方案显示 `☑` 标记
- 新增替换模式切换，按 `M` 可在“单向复制”和“双向交换”之间切换
- 默认替换模式改为“单向复制”
- 新增与替换模式、当前配置匹配相关的测试覆盖

### Changed
- `Enter` 执行成功后直接刷新当前候选页，不再进入结果详情页
- 只有执行失败时才进入结果详情页
- 顶部标题中的模式文案改为“当前模式”
- 调整 TUI 标记字符，避免因字符宽度问题导致预览面板错位
- 优化滚动条和光标显示，减少内容一致时的视觉噪音

### Internal
- 将替换模式抽象为计划层和执行层的显式类型
- 将当前配置匹配判定和执行后视图切换提取为独立逻辑，降低 UI 耦合
